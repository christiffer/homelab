List current homelab tasks from Vikunja using the CLI:

```bash
.vikunja/vk list
```

After listing, offer to help with any tasks or create new ones. Use `.vikunja/vk` for all task operations:
- `vk show <id>` - view details
- `vk create "title" --priority=P2` - create task
- `vk done <id>` - mark complete
- `vk update <id> --title="..."` - update task
